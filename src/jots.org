If you put a =:DATE:= property, it will be shown in the exported HTML. \\
You can also set =:CUSTOM_ID:= if you don't want id to be derived from the jot heading.

* The idea of a jot
:PROPERTIES:
:DATE:     2026-05-01
:END:
Jot is when you want to write something more casual and shorter than a post. "microblogging".
